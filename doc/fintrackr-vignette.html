Using fintrackr for Personal Finance Analysis

Introduction

The fintrackr package provides tools for personal finance management, including budgeting, investment growth analysis, debt repayment projection, strategy comparison, and financial visualization.

Example Data

finance_data <- data.frame(
  date = as.Date(c("2026-01-01", "2026-01-15", "2026-02-01", "2026-02-10")),
  category = c("Salary", "Groceries", "Salary", "Rent"),
  income = c(3000, 0, 3000, 0),
  expense = c(0, 200, 0, 1200)
)
finance_data
##         date  category income expense

## 2 2026-01-15 Groceries      0     200

This function helps users summarize their financial activity by category and by month, while also computing total income, total expenses, and net balance.
